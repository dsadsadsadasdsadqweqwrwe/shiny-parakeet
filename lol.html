Operating Systems (Safe / Privacy / Security)

Qubes OS: Max isolation. Compartmentalized VMs.

Kali Linux: Pen-testing toolkit, preloaded tools.

Parrot OS: Lightweight security OS.

Tails OS: Tor‑based amnesia system, leaves no traces.

Recommended Specs (for Cybersecurity Work)

CPU: 6+ cores (Ryzen 5/7, i5/i7)

RAM: 16GB minimum, 32GB ideal (VMs, Kali, Qubes)

Storage: 512GB SSD minimum

GPU: Optional unless cracking hashes (RTX cards help)

Private Browsers

Firefox Hardened: Best balance of privacy & usability.

LibreWolf: Debloated, hardened Firefox.

Tor Browser: For anonymity (slow, but safe).

Recommended Browser Extensions

uBlock Origin: Blocks ads, trackers.

Privacy Badger: Blocks invisible trackers.

NoScript (advanced): Blocks all scripts (requires knowledge).

HTTPS Everywhere: Forces encryption.
